## Overview

APGI predicts that the threshold and signal dynamics exhibit
long-range temporal correlations consistent with 1/f power spectra —
a signature of hierarchical multi-timescale systems (Spec §12).

This guide explains how to validate the spectral signature of your APGI system.

## Quick Start

```python
import numpy as np
from stats.spectral_model import validate_spectral_signature

# Run APGI simulation and collect signal history
outputs = []
for t in range(10000):
    output = pipeline.step(x_e_t, x_i_t)
    outputs.append(output)

# Extract signal history
S_history = np.array([o['S'] for o in outputs])

# Validate spectral signature
result = validate_spectral_signature(
    S_history,
    taus=pipeline.taus if hasattr(pipeline, 'taus') else None,
    dt=1.0
)

print(result['message'])
# Output: "Spectral exponent β=1.05 (Hurst H=0.53).
# Healthy range: [0.8, 1.5]. ✅ Valid"
```

## Spectral Theory

### Power Spectral Density (1/f Structure)

The power spectrum of the threshold process generated by a hierarchy
of L relaxation oscillators:

```text
S_θ(f) = Σ_ℓ σ²_ℓ · τ²_ℓ / [1 + (2πfτ_ℓ)²]
```

Each term is a Lorentzian. Their superposition produces an
approximate 1/f^β power law over the decade range
[1/τ_{L-1}, 1/τ_0].

### Spectral Exponent

The empirical spectral exponent is estimated as the log-log slope:

```text
β_spec = -d log P(f) / d log f
```

**Healthy range:** β_spec ∈ [0.8, 1.5]

- β < 0.8: White noise (no long-range correlations)
- β ≈ 1.0: Pink noise (1/f, optimal)
- β > 1.5: Brown noise (over-correlated)

### Hurst Exponent

For fractional Brownian motion and related long-range correlated processes:

```text
H ≈ (β_spec + 1) / 2
```

**Interpretation:**

- H = 0.5: Random walk (no memory)
- H ∈ (0.5, 1): Long-range dependence (persistent)
- H = 1.0: Brownian motion

## Validation Methods

### Method 1: Spectral Exponent Estimation

Estimate the spectral exponent from the power spectrum:

```python
from stats.spectral_model import estimate_1f_exponent

# Compute power spectrum
freqs, power = scipy.signal.welch(S_history, fs=1.0/dt)

# Estimate spectral exponent
beta_spec = estimate_1f_exponent(freqs, power)

print(f"Spectral exponent: β = {beta_spec:.2f}")

# Check if in healthy range
if 0.8 <= beta_spec <= 1.5:
    print("✅ Spectral signature is healthy")
else:
    print("⚠️ Spectral signature outside healthy range")
```

### Method 2: Lorentzian Superposition Fitting

Fit the theoretical Lorentzian superposition to the observed spectrum:

```python
from stats.spectral_model import fit_lorentzian_superposition
import scipy.signal

# Compute power spectrum
freqs, power = scipy.signal.welch(S_history, fs=1.0/dt, nperseg=512)

# Get timescales from hierarchical system
taus = pipeline.taus if hasattr(pipeline, 'taus') else np.array([10, 16, 26, 42])

# Fit Lorentzian superposition
result = fit_lorentzian_superposition(freqs, power, taus)

print(f"R² = {result['r_squared']:.3f}")
print(f"Amplitudes: {result['amplitudes']}")

# Good fit if R² > 0.8
if result['r_squared'] > 0.8:
    print("✅ Lorentzian superposition fits well")
else:
    print("⚠️ Poor fit to Lorentzian model")
```

### Method 3: Hurst Exponent Estimation

Estimate the Hurst exponent using rescaled range analysis:

```python
from stats.hurst import estimate_hurst_robust

# Estimate Hurst exponent
H = estimate_hurst_robust(S_history)

print(f"Hurst exponent: H = {H:.2f}")

# Compute implied spectral exponent
beta_spec = 2 * H - 1

print(f"Implied spectral exponent: β = {beta_spec:.2f}")

# Check if in healthy range
if 0.8 <= beta_spec <= 1.5:
    print("✅ Hurst exponent indicates healthy dynamics")
else:
    print("⚠️ Hurst exponent outside healthy range")
```

### Method 4: Complete Validation

Use the comprehensive validation function:

```python
from stats.spectral_model import validate_spectral_signature

result = validate_spectral_signature(
    S_history,
    taus=pipeline.taus,
    dt=1.0,
    tolerance=0.1
)

print(result['message'])
print(f"Valid: {result['valid']}")
print(f"Spectral exponent: {result['spectral_exponent']:.2f}")
print(f"Hurst exponent: {result['hurst_exponent']:.2f}")
print(f"Lorentzian fit R²: {result['lorentzian_fit']['r_squared']:.3f}")
```

## Interpretation

### Healthy Spectral Signature

```text
Spectral exponent β ∈ [0.8, 1.5]
Hurst exponent H ∈ [0.4, 0.75]
Lorentzian fit R² > 0.8
```

**Indicates:**

- ✅ Long-range temporal correlations
- ✅ Hierarchical multi-timescale dynamics
- ✅ Optimal information processing

### Unhealthy Spectral Signature

**β < 0.8 (White noise):**

- Indicates: No long-range correlations
- Cause: Insufficient hierarchical structure
- Fix: Enable hierarchical system or increase number of levels

**β > 1.5 (Brown noise):**

- Indicates: Over-correlated dynamics
- Cause: Excessive coupling or slow timescales
- Fix: Reduce coupling strengths or increase timescale ratio k

**Poor Lorentzian fit (R² < 0.8):**

- Indicates: Dynamics don't match hierarchical model
- Cause: Non-hierarchical noise or model mismatch
- Fix: Check configuration; verify hierarchical system is enabled

## Configuration Effects

### Effect of Number of Levels

```python
# 2 levels: Limited spectral range
config = {'hierarchical_mode': 'full', 'n_levels': 2}
# Result: β ≈ 0.6 (white noise)

# 4 levels: Good spectral range
config = {'hierarchical_mode': 'full', 'n_levels': 4}
# Result: β ≈ 1.0 (pink noise) ✅

# 8 levels: Extended spectral range
config = {'hierarchical_mode': 'full', 'n_levels': 8}
# Result: β ≈ 1.2 (slightly brown)
```

### Effect of Timescale Ratio

```python
# k = 1.2 (small ratio): Closely spaced timescales
config = {'hierarchical_mode': 'full', 'k': 1.2}
# Result: β ≈ 0.9 (good)

# k = 1.6 (default): Well-spaced timescales
config = {'hierarchical_mode': 'full', 'k': 1.6}
# Result: β ≈ 1.0 (optimal) ✅

# k = 2.0 (large ratio): Widely spaced timescales
config = {'hierarchical_mode': 'full', 'k': 2.0}
# Result: β ≈ 1.1 (good)
```

### Effect of Coupling Strength

```python
# Weak coupling: Limited cross-level interaction
config = {'hierarchical_mode': 'advanced', 'C_down': 0.01, 'C_up': 0.01}
# Result: β ≈ 0.8 (white noise)

# Moderate coupling: Balanced interaction
config = {'hierarchical_mode': 'advanced', 'C_down': 0.1, 'C_up': 0.05}
# Result: β ≈ 1.0 (pink noise) ✅

# Strong coupling: Excessive interaction
config = {'hierarchical_mode': 'advanced', 'C_down': 0.5, 'C_up': 0.3}
# Result: β ≈ 1.3 (brown noise)
```

## Practical Examples

### Example 1: Validate Single-Scale System

```python
import numpy as np
from stats.spectral_model import validate_spectral_signature

# Single-scale system
config = {'hierarchical_mode': 'off'}
pipeline = APGIPipeline(config)

# Run simulation
outputs = []
for t in range(10000):
    output = pipeline.step(x_e_t, x_i_t)
    outputs.append(output)

S_history = np.array([o['S'] for o in outputs])

# Validate
result = validate_spectral_signature(S_history, dt=1.0)
print(result['message'])
# Expected: β ≈ 0.5 (white noise, no hierarchy)
```

### Example 2: Validate Hierarchical System

```python
# Hierarchical system
config = {
    'hierarchical_mode': 'full',
    'n_levels': 4,
    'tau_0': 10.0,
    'k': 1.6,
}
pipeline = APGIPipeline(config)

# Run simulation
outputs = []
for t in range(10000):
    output = pipeline.step(x_e_t, x_i_t)
    outputs.append(output)

S_history = np.array([o['S'] for o in outputs])

# Validate
result = validate_spectral_signature(S_history, taus=pipeline.taus, dt=1.0)
print(result['message'])
# Expected: β ≈ 1.0 (pink noise, healthy hierarchy)
```

### Example 3: Optimize Configuration

```python
import numpy as np
from stats.spectral_model import validate_spectral_signature

# Test different configurations
configs = [
    {'hierarchical_mode': 'basic', 'n_levels': 3},
    {'hierarchical_mode': 'basic', 'n_levels': 4},
    {'hierarchical_mode': 'basic', 'n_levels': 5},
    {'hierarchical_mode': 'advanced', 'n_levels': 4, 'C_down': 0.05},
    {'hierarchical_mode': 'advanced', 'n_levels': 4, 'C_down': 0.1},
    {'hierarchical_mode': 'advanced', 'n_levels': 4, 'C_down': 0.2},
]

results = []
for config in configs:
    pipeline = APGIPipeline(config)
    
    # Run simulation
    outputs = []
    for t in range(10000):
        output = pipeline.step(x_e_t, x_i_t)
        outputs.append(output)
    
    S_history = np.array([o['S'] for o in outputs])
    
    # Validate
    result = validate_spectral_signature(S_history, taus=pipeline.taus, dt=1.0)
    results.append({
        'config': config,
        'beta': result['spectral_exponent'],
        'valid': result['valid'],
    })

# Find best configuration
best = max(results, key=lambda r: r['valid'] and abs(r['beta'] - 1.0))
print(f"Best configuration: {best['config']}")
print(f"Spectral exponent: β = {best['beta']:.2f}")
```

## Troubleshooting

### Issue: Spectral exponent is too low (β < 0.8)

**Symptoms:** White noise spectrum, no long-range correlations

**Causes:**

1. Hierarchical system not enabled
2. Too few levels
3. Timescale ratio too large

**Solutions:**

```python
# Enable hierarchical system
config = {'hierarchical_mode': 'full'}

# Increase number of levels
config = {'n_levels': 6}

# Reduce timescale ratio
config = {'k': 1.4}
```

### Issue: Spectral exponent is too high (β > 1.5)

**Symptoms:** Brown noise spectrum, over-correlated dynamics

**Causes:**

1. Coupling strengths too high
2. Decay timescale too long
3. Too many levels

**Solutions:**

```python
# Reduce coupling strengths
config = {'C_down': 0.05, 'C_up': 0.02}

# Reduce decay timescale
config = {'tau_pi': 500.0}

# Reduce number of levels
config = {'n_levels': 3}
```

### Issue: Lorentzian fit is poor (R² < 0.8)

**Symptoms:** Observed spectrum doesn't match theoretical prediction

**Causes:**

1. Non-hierarchical noise sources
2. Model mismatch
3. Insufficient data

**Solutions:**

```python
# Increase simulation length
for t in range(100000):  # 10x longer
    output = pipeline.step(x_e_t, x_i_t)

# Verify hierarchical system is enabled
config = {'hierarchical_mode': 'full'}

# Check for external noise sources
# Reduce input noise if present
```

## Advanced Topics

### Spectral Exponent vs. Hurst Exponent

The relationship between spectral exponent and Hurst exponent:

```text
H = (β + 1) / 2
```

**Conversion table:**

| β  | H    | Type  |
|----|------|-------|
| 0.0| 0.5  | White |
| 0.5| 0.75 | Pink  |
| 1.0| 1.0  | Brown |
| 1.5| 1.25 | Brown |

### Multifractal Analysis

For advanced analysis, use multifractal detrended fluctuation analysis (MFDFA):

```python
from stats.hurst import estimate_hurst_robust

# MFDFA can detect non-stationary dynamics
H_mfdfa = estimate_hurst_robust(S_history, method='mfdfa')
```
